Техническое задание: Landing Page для Bird Game 3 Token

1. Проект

Bird Game 3 — мемный криптопроект, основанный на вирусном TikTok-тренде о несуществующей MOBA-игре 5v5. Весь контент (геймплей, турниры, персонажи-птицы) создан ИИ и является частью мема.

Суть: запуск мем-токена как «внутриигровой валюты» несуществующей игры.
Токен: $GRAIN (или альтернативное название).

⸻

2. ЦА
	•	TikTok-аудитория, знакомая с мемом
	•	Криптоэнтузиасты и фанаты мемкоинов
	•	MOBA-геймеры
	•	RU + EN (приоритет RU)
	•	Возраст: 16–35

⸻

3. Цели сайта
	•	Быстро привлечь внимание (жизнь тренда 1–2 недели)
	•	Объяснить мем и идею токена
	•	Создать FOMO через роадмап
	•	Дать понятный путь к покупке
	•	Сформировать комьюнити

⸻

4. Формат сайта

Одностраничный лендинг, адаптивный, с быстрым загрузом.

Основные секции:
	1.	Hero — заголовок, подзаголовок про $GRAIN, CTA «Buy», фон-видео из TikTok
	2.	About — что такое Bird Game 3 и зачем токен
	3.	Lore — шуточная история «разработки», Bird Game 1/2
	4.	Tokenomics — supply, распределение, адрес контракта, где купить
	5.	Roadmap — юмористичный «релиз игры» + реальные шаги токена
	6.	Community / TikTok Gallery — видео, мемы, скриншоты
	7.	How to Buy — пошаговая инструкция, DEX, кошельки
	8.	Join the Flock — TikTok, Telegram, X, Discord
	9.	Footer — дисклеймеры, риски, контакты

⸻

5. Функционал

Обязательно:
	•	Адаптив
	•	Быстрая загрузка (<3 сек)
	•	Кнопки «Buy»
	•	Копирование адреса контракта
	•	Таймер «релиза игры»
	•	TikTok-галерея

Опционально:
	•	Анимации при скролле
	•	Мини-игра / квиз
	•	Airdrop-форма
	•	Live-цена токена
	•	FAQ-бот

⸻

6. Дизайн
	•	Стиль: геймерский + мемный + киберспорт
	•	Цвета: тёмный фон, неон (cyan/green), золото для токена
	•	Шрифты: Orbitron / Exo для заголовков, Inter / Roboto для текста
	•	Графика: птицы-персонажи, мемы, glow-эффекты, частицы
	•	Анимации: hover, плавные переходы, вращение токенов

⸻

7. Контент
	•	Языки: EN (+ RU опционально)
	•	Ироничный тон
	•	Tokenomics, roadmap (3–5 этапов), FAQ (10–15), дисклеймер
	•	Медиа: 10–15 TikTok-видео, 20+ мемов, логотипы

⸻

8. Технологии
	•	HTML/CSS (Tailwind) + JS
	•	React / Next.js (по желанию)
	•	GSAP / Anime.js
	•	Хостинг: Vercel / Netlify
	•	Оптимизация: WebP, lazy load, CDN
	•	SEO, OG-теги, аналитика

⸻

9. Сроки
	•	Подготовка: 1–2 дня
	•	Разработка: 3–5 дней
	•	Тесты + запуск: 2–3 дня
Итого: 6–10 дней

⸻

10. Важно
	•	Проект мемный, Bird Game 3 не существует
	•	Не инвестиционная рекомендация
	•	Privacy Policy, Terms, Cookie notice

⸻

11. Критерии успеха
	•	<3 сек загрузка
	•	1000+ посетителей за неделю
	•	Конверсия >5%
	•	Время на сайте >1 мин
